# FixSubtitles
Sometimes subtitles come with some extra numbers below the main text.
This very simple utility helps to clear them out.
## The usage is very simple:
1. Choose whether you want to process directories recursively. If you want recursion, then check `Include subdirectories` button. 
2. Press `Fix Subtitles` button.
3. Select desired directory.
4. Upon finishing you will see message box `Well done!`.
